**The Classical Cinema Archive**

A minimalist, elegant digital museum for the greatest films ever made. Built with Next.js 16, TypeScript, and Tailwind CSS.

## Features

- **Cinematic Design**: Noir/Sepia aesthetic with *Playfair Display* typography.
- **Movie Collection**: A curated list of masterpieces (Fellini, Hitchcock, Kubrick).
- **Detail Views**: Deep dives into each film with synopsis, cast, and metadata.
- **Responsive**: Fully responsive grid and hero layouts.

## Tech Stack

- **Framework**: Next.js 16.1.1 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS v4
- **Icons**: Lucide React

## Getting Started

1. **Install Dependencies**
   ```bash
   npm install
   ```

2. **Run Development Server**
   ```bash
   npm run dev
   ```

3. **Open Browser**
   Navigate to [http://localhost:3000](http://localhost:3000).

## Project Structure

- `app/`: Next.js App Router pages.
- `components/`: UI components (Hero, MovieCard, NavBar).
- `lib/`: Mock data (`movies.ts`) and utilities.
- `globals.css`: Tailwind v4 configuration and theme variables.
